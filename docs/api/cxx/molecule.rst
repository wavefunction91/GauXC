Molecule and atom types
=======================

These APIs describe the basic molecular data structures used throughout GauXC.
The :cpp:class:`GauXC::Molecule` type is a lightweight container of :cpp:struct:`GauXC::Atom` instances, providing methods for handling a collection of atoms.
